# Trading Card Manager

画像付きでカードを管理し、出品・統計・バックアップまで行える  
個人向けトレーディングカード管理アプリです。

---

## 概要

Trading Card Manager は、トレーディングカードを  
画像付きで管理し、フリマ出品や統計表示まで行える  
**完全ローカル完結型のカード管理アプリ**です。

アカウント登録やログインは不要で、  
すべてのデータは端末内にのみ保存されます。

---

## 主な機能

- カード情報の登録・編集・削除
- 画像付きカード管理（端末内保存）
- タグ・検索・フィルタ機能
- フリマ出品管理（出品中／売却済み）
- 統計表示（年別・月別）
- データのエクスポート／インポート（JSON）

---

## バックアップと復元

### データのエクスポート（バックアップ）

- アプリ内の「設定」画面からデータを **JSON形式** でエクスポートできます
- Share機能を使って Files アプリ等に保存してください

### データの保存について

- バックアップファイル（JSON）の管理はユーザー自身で行ってください
- アプリを削除すると端末内データは失われます

### データの復元（インポート）

- 保存した JSON ファイルを選択することで復元できます
- カードデータ／出品データを復元可能です

## 使い方
1. カードを追加する
	1.	画面右上の「＋」アイコンをタップ
	2.	カード名・説明・価格などを入力
	3.	必要に応じて画像・タグ・取得日を設定
	4.	「保存」をタップして登録完了
2. カードを編集・削除する
	•	編集
	1.	カード一覧から対象のカードをタップ
	2.	「編集」を選択
	3.	内容を変更して保存
	•	削除
	1.	カード一覧でカードを左にスワイプ
	2.	削除確認ダイアログで「削除」を選択
3. フリマ出品を管理する
	1.	カード詳細画面で「出品」をタップ
	2.	販売価格・出品先・説明文を入力
	3.	保存すると「出品中」として管理されます
	4.	売却後は「売却済み」に変更できます
　※ 出品中のカードは一覧画面でバッジ表示されます
4. 統計を確認する
	1.	一覧画面の統計アイコンをタップ
	2.	月別・年別の金額推移を確認できます
	3.	コレクションの成長を可視化できます
5. バックアップを取る（重要）
　	1.	設定画面を開く
	2.	「エクスポート」を選択
	3.	Shareシートから「ファイルに保存」を選択
	4.	Filesアプリ内にバックアップを保存

### ⚠️ 注意事項（重要）

- **画像データはバックアップ対象外です**
- 復元後、画像は表示されません（UI上で明示されます）

## プライバシーポリシー

### 収集する情報

- 本アプリは **個人情報を一切収集しません**
- アカウント登録、ログイン機能はありません

### データの保存について

- すべてのデータは端末内にのみ保存されます
- 外部サーバーへの送信は一切行いません

### 第三者提供

- 取得したデータを第三者へ提供することはありません
