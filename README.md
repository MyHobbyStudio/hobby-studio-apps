## Trading Card Manager
個人向けのトレーディングカード管理アプリです。

### 主な機能
- カード情報の登録・編集・削除
- 画像付きカード管理（端末内保存）
- フリマ出品管理
- 統計表示（年別・月別）
- データのエクスポート／インポート（JSON）

---

## プライバシーポリシー
本アプリは以下の方針で設計されています。

### 収集する情報
- 本アプリは**個人情報を一切収集しません**
- アカウント登録、ログイン機能はありません

### データの保存について
- すべてのデータは**端末内にのみ保存**されます
- 外部サーバーへの送信は行いません

### バックアップ機能について
- ユーザー操作によりJSONファイルとしてエクスポートできます
- バックアップファイルの管理はユーザー自身の責任で行ってください
- 画像データはバックアップ対象外です

### 第三者提供
- データを第三者へ提供することはありません

### お問い合わせ
- 本アプリは個人開発のため、お問い合わせ窓口は設けていません
